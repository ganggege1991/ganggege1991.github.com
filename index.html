在浏览器的移动端区分iPhone，Android

业务需求:需要通过压面下载app，Android下载Android的app，iPhone则跳到AppStore。
解决方案:通过浏览器的navigator对象，可以获取到设备，操作系统的一些相关的信息(包括系统版本号)。通过这些信息则可以做出判断。

首先

打印出navidator对象，分析一下。

如下

var s = navigator.userAgent;
if(s.indexOf("Android")>0){
 //Android;
}
else if(s.indexOf("iPhone")>0){
 //iPhone;
}

实现响应式的表格

By He Gang

发表于 2015-07-11

如何在移动端更好显示表格

今天看到，看到“堆叠”的方式显示表格，瞬间感觉高大上，纯css实现。然后对他的实现方式研究了一下。

当屏幕打到需要”堆叠”方式显示的时候，则更换样式。类似于sql中的行转列的效果。

td中需要一个属性(这里是data-label)，记录列属性的值。

<td data-label="Payment">Payment #1</td>
<td data-label="Issue Date">02/01/2015</td>
<td data-label="Amount">$2,311</td>
<td data-label="Period">01/01/2015 - 01/31/2015</td>

css

@media screen and (max-width: 600px) {

table {
  border: 0;
}
/*隐藏 thead*/
table thead {
  display: none;
}

table tr {
  margin-bottom: 10px;
  display: block;
  border-bottom: 2px solid #ddd;
}

table td {
/*改变td的显示方式，为block，则前后会有换行效果*/
  display: block;
  text-align: right;
  font-size: 13px;
  border-bottom: 1px dotted #ccc;
}

table td:last-child {
  border-bottom: 0;
}
/*在td前显示原本列名*/
table td:before {
  /*访问，td的data-label的属性*/
  content: attr(data-label);
  float: left;
  text-transform: uppercase;
  font-weight: bold;
}
}
